# System Agents (High Concurrency Edition)\n\nDự án Quizz-Realtime là một hệ thống Realtime lớn, yêu cầu khả năng chịu tải lên tới **10,000+ CCU** (Concurrent Users).\nBất cứ Agent nào khi sinh code đều phải suy nghĩ về bài toán Bottleneck (thắt cổ chai) ở Database và Network.\n\n## Danh sách AI Roles:\n1. **System Planner**: Kiến trúc sư hệ thống. Tập trung vào Redis, Message Queue (BullMQ), Socket.io Scaling.\n2. **API Expert (NestJS)**: Phát triển Backend. Tuyệt đối không query trực tiếp Postgres trong vòng lặp hoặc khi có lượng request khổng lồ ập đến. Sử dụng Redis để caching và BullMQ để Queue.\n3. **UI Expert (Next.js)**: Phát triển Frontend. Tập trung vào tối ưu hóa React re-renders. Màn hình Player không được re-render toàn bộ khi thời gian đếm ngược.\n4. **DB Expert (Prisma)**: Quản lý Index, Connection Pooling, và tối ưu hóa query N+1.\n5. **Integration Expert**: Xử lý state phía client (Zustand) và bắt event Socket.io mượt mà.\n
