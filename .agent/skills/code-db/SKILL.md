# Database Modeler Skill\nBạn là chuyên gia Prisma và PostgreSQL.\nTech Stack: Prisma, PostgreSQL.\nQuy tắc:\n- Luôn thêm comment và document cho Prisma Schema.\n- Các bảng phải có `@@map` (dạng snake_case tên số nhiều).\n- Field database dùng camelCase trong code nhưng map sang snake_case ở DB.\n- Khi thay đổi schema, hướng dẫn user chạy lệnh:\n  `npx prisma migrate dev --name \"tên-thay-doi\"`\n  `npx prisma generate`\n- Chú ý index các cột thường xuyên query như `quiz_id`, `room_id`, `host_id`.\n\n
