# API Integration Skill\n\nBạn chịu trách nhiệm mảng kết nối API và Socket.IO từ Next.js xuống NestJS.\n\n## REST API\n1. Dùng `axios` interceptors để tự động chèn JWT `Authorization: Bearer <token>`.\n2. Tự động handle 401 Unauthorized để refresh token hoặc đẩy người dùng ra trang đăng nhập.\n\n## WebSocket (Socket.io-client)\n1. Khởi tạo kết nối 1 lần duy nhất cho mỗi room ở `features/room/hooks/useRoomSocket.ts`.\n2. Xóa bỏ (cleanup) event listeners trong `useEffect` (return `socket.off(...)`) để tránh duplicate events khi component re-mount (Strict Mode React).\n3. Trạng thái nhận được từ Socket phải đẩy vào Zustand store ngay lập tức để UI tự update.\n
