# Luồng Batch Insert Answers (BullMQ)\n\n1. Player gọi Socket/API `submit_answer(roomId, questionId, answerId)`.\n2. NestJS Controller -> gọi `QueueService.addJob('answerQueue', { payload })`.\n3. NestJS trả về HTTP 202 Accepted (hoặc ack socket) ngay lập tức (response time < 10ms).\n4. BullMQ Processor gom 500 job/lần.\n5. Processor tính điểm -> Cập nhật Redis Leaderboard -> Build mảng dữ liệu -> `prisma.playerAnswer.createMany`.\n6. Nếu DB sập tạm thời, BullMQ tự động retry, không mất dữ liệu.\n
