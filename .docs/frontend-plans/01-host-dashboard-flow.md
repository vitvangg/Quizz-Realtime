# Luồng Host Dashboard\n\n1. **Tạo phòng**: Host chọn Quiz -> `POST /rooms` -> Server trả về PIN (ví dụ `123456`).\n2. **Lobby**: Host render trang `room/123456` (chế độ Host).\n   - Lắng nghe event `player_joined` -> thêm vào danh sách.\n3. **Start Game**: Bấm Start -> Gọi API -> Nhận event `game_started` -> Chuyển giao diện sang màn hình câu hỏi 1.\n4. **Question Screen**: \n   - Hiển thị Nội dung câu hỏi + Biểu đồ số người đã trả lời (Realtime qua socket event `answer_count_updated`).\n   - Hết giờ -> Hiển thị đáp án đúng + Leaderboard Top 5 (lấy từ Redis).\n
